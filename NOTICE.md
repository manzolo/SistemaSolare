## Astronomical data and factual references

- Approximate planetary positions and Keplerian elements:
  [NASA/JPL Solar System Dynamics, Approximate Positions of the Planets](https://ssd.jpl.nasa.gov/planets/approx_pos.html)
- Planetary satellite and small-body orbital data:
  [NASA/JPL Solar System Dynamics](https://ssd.jpl.nasa.gov/)
- Ephemeris reference:
  [JPL Horizons](https://ssd.jpl.nasa.gov/horizons/)
- Names and recognition status of planetary satellites:
  [International Astronomical Union](https://www.iau.org/public/themes/naming/)
- General planetary, asteroid and comet descriptions:
  [NASA Solar System Exploration](https://science.nasa.gov/solar-system/)
- Rosetta and comet 67P mission context:
  [European Space Agency - Rosetta](https://www.esa.int/Science_Exploration/Space_Science/Rosetta)

Numerical values and descriptions are educational summaries. Orbits, sizes and
distances are visually rescaled, and the application must not be used as a
precision ephemeris, navigation tool or authoritative scientific catalogue.

The project identifies the principal sources used to verify factual values.
Descriptive text in the application is original paraphrasing rather than copied
publication text.

## Music

The musical feature is a code-generated, original synthetic arrangement
inspired by J.S. Bach's *Air* from Orchestral Suite No. 3, BWV 1068. The
underlying composition is in the public domain. The project does not contain or
reproduce a third-party audio recording.

## Privacy and external requests

The project contains no analytics, advertising, tracking pixels or first-party
cookies. It uses browser `localStorage` only to remember the selected language
and `sessionStorage` only to coordinate view transitions.

The current single-file version loads fonts from Google Fonts and planet images
from NASA-hosted servers. A visitor's browser therefore makes ordinary HTTPS
requests to those third-party services, which may receive standard connection
data such as IP address, user agent and referrer according to their own privacy
policies:

- Google Privacy Policy: https://policies.google.com/privacy
- NASA Privacy Policy: https://www.nasa.gov/privacy/

For deployments requiring all assets to remain first-party, the open-source
font files and permitted image copies can instead be self-hosted while
preserving their licenses and credit lines.
